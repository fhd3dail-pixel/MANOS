# 🤖 MANOS - منصة ذكية للاختبار الأخلاقي

**MANOS** هي منصة متقدمة تجمع بين الذكاء الاصطناعي المتطور وأدوات اختبار الشبكات والـ Web Scraping.

## 🎯 المميزات الرئيسية

- 🧠 **ذكاء اصطناعي متقدم** - يتحدث البغدادية بطلاقة
- 📱 **تطبيق APK كامل** - واجهة احترافية على الأندرويد
- 🌐 **اختبار الشبكات** - للشبكات المحلية فقط (أخلاقي)
- 🕷️ **Web Scraping** - جمع البيانات بذكاء
- ⚡ **أداء عالي جداً** - سرعة فائقة 100%
- 🔐 **آمن وموثوق** - للتعليم والاختبار فقط

## 📋 المتطلبات

- Python 3.11+
- Kotlin 1.9+
- Android SDK
- Node.js (اختياري)

## 🚀 البدء السريع

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend (APK)
```bash
cd frontend
./gradlew build
```

## 📂 هيكل المشروع

```
MANOS/
├── backend/              # الخادم الرئيسي (Python FastAPI)
├── frontend/             # تطبيق APK (Kotlin)
├── ai-models/            # نماذج الذكاء الاصطناعي
├── tools/                # أدوات اختبار الشبكات
└── docs/                 # التوثيق
```
